# 作業 4 表單互動

## 作業說明

設計四個題目來複習有關表單的使用。暫時不使用資料庫，每一個題目都只是上傳一些訊息，然後由 php 接受表單訊息後，做出回應。你可以將表單跟回應設計在同一個檔案，例如 f01.php 或是將輸入與回應分開，例如 f01.html + f01.php。

檔案名稱分別為 f01, f02, f03, f04 。
