# 作業 4 表單互動

## 作業說明

設計四個題目來複習有關表單的使用。暫時不使用資料庫，每一個題目都只是上傳一些訊息，然後由 php 接受表單訊息後，做出回應。你可以將表單跟回應設計在同一個檔案，例如 f01.php 或是將輸入與回應分開，例如 f01.html + f01.php。

檔案名稱分別為 f01, f02, f03, f04 。

1. 設計一個表單可以輸入姓名、帳號、出生年月日、性別、喜好。設計 php 程式碼根據輸入的性別及出生年月日，計算出使用者的幸運色，並回應給使用者。 (檢查輸入的年月日是否合理，若不合理，回應輸入不合法。)
2. 第二題的說明
3. 設計一個活動報名表單，讓使用者輸入基本資料，PHP 接收後檢查必填欄位、Email 格式與出生年月日是否合法，並依照年齡與經驗判斷是否符合報名資格，最後顯示審核結果。
